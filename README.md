# 🎬 YouTube Summary Bot

Telegram бот который пересказывает YouTube видео с помощью AI.

## Что умеет
- Принимает ссылку на YouTube видео
- Достаёт субтитры
- Делает краткое резюме на русском языке с помощью AI

## Технологии
- Python
- python-telegram-bot
- Groq AI (llama-3.3-70b)
- youtube-transcript-api

## Как запустить
1. Установи библиотеки: `pip install python-telegram-bot groq youtube-transcript-api`
2. Вставь свой BOT_TOKEN и GROQ_KEY в код
3. Запусти: `python bot.py`
